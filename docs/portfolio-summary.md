# Portfolio Summary

## 프로젝트 소개

기존 데이터 검증 및 운영 자동화 경험을 금융 데이터 처리 흐름으로 확장하기 위한 Java/Spring 기반 백엔드 프로젝트다. 외부 거래 CSV 검증, 내부 원장 대사, 일별 정산과 감사 가능한 운영 이력을 하나의 서비스로 모델링한다.

## 현재 확인 가능한 내용

- PostgreSQL 관계형 모델과 Flyway 마이그레이션 기준선 설계
- 파일 중복, 거래 고유성, 정산 일자 고유성 등 데이터 정합성 제약 설계
- API 응답/예외 처리와 CI 기반 개발 시작점 구성

## 기능 구현 후 보강할 지표

- 업로드/대사/정산 API의 테스트 시나리오와 통과 결과
- 대량 CSV 처리 시간과 메모리 사용량
- 배치 재실행 및 실패 복구 검증 결과
